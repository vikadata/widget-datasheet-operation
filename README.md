# 维格小程序 - 字段编辑器（Field Editor）

用于测试小程序关于字段编辑的 SDK 能力

## 快速开始

### 安装依赖

```shell
yarn install
```

### 启动

```shell
widget-cli start
```

## 特性

### 已经实现的功能特性

- [x] 修改所选字段的属性配置
- [x] 修改所选字段的描述
- [x] 新增单个字段
- [x] 删除单个字段
- [x] 一键「新增/删除」多个字段
